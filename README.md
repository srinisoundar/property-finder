# Property-finder

Property finder app

Make sure you're using the latest version of Node.js and NPM

### Quick start

```bash
# install the dependencies with npm
$ npm install

# start
$ npm start
```

go to [http://localhost:8080](http://localhost:8080) in your browser.

###Production 

```bash
# build
$ npm run build
```

### Run test

```bash
#test
$ npm run test
```
